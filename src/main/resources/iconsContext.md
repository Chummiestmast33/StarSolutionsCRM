# Contexto de Librería de Iconos: FluentUI

Esta librería de iconos se divide en dos estilos (Filled y Regular) y en dos bloques alfabéticos (A-L y M-Z).
Las clases CSS de los iconos se construyen algorítmicamente tomando el nombre base del icono (Enum), convirtiéndolo a minúsculas, reemplazando los guiones bajos (`_`) por guiones medios (`-`), y anteponiéndole un prefijo específico según su clasificación.

## Reglas de Mapeo de Prefijos

| Estilo | Rango Alfabético | Prefijo | Ejemplo de Enum Original | Clase CSS Generada |
| :--- | :--- | :--- | :--- | :--- |
| Filled | A-L | `fltfal-` | `ACCESSIBILITY_16` | `fltfal-accessibility-16` |
| Filled | M-Z | `fltfmz-` | `MAGIC_WAND_16` | `fltfmz-magic-wand-16` |
| Regular| A-L | `fltral-` | `APP_RECENT_24` | `fltral-app-recent-24` |
| Regular| M-Z | `fltrmz-` | `ZOOM_IN_20` | `fltrmz-zoom-out-20` |

## Instrucciones de Inferencia para la IA:
1. Identifica el concepto del icono que el usuario necesita (ej. "calendar").
2. Revisa la letra inicial ("C" cae en el rango A-L).
3. Selecciona el estilo indicado por el usuario (Filled o Regular) y aplica el prefijo correspondiente.
4. Anexa el sufijo de tamaño estándar requerido. Los tamaños estandarizados más comunes que acompañan a la raíz del icono son `_12`, `_16`, `_20`, `_24`, `_28`, `_32`, y `_48`.
5. Resultado final esperado: `fltfal-calendar-24`.
